## Acerca del proyecto
Este proyecto es una tarea del grado de Desarrollo de Aplicaciones Multiplataforma para la asignatura de 
Desarrollo y Despliegue en Entornos Colaborativos y en la Nube.

La tarea consiste en familiarizarse con los conceptos básicos del control de versiones con Git y GitHub, realizando 
mejoras a un archivo HTML por defecto e implementándo nuevas funcionalidades creando diferentes ramas.

## Instalación

La tarea consta de dos páginas HTML relacionadas entre ellas. Existen varias opciones para previsualizarlas, aunque 
aquí se incluye la forma más sencilla que es vía navegador web o con la extensión [Live Preview](https://marketplace.visualstudio.com/items?itemName=ms-vscode.live-server) de VSCODE

### Clonar el repositorio

   ```sh
   git clone https://github.com/little-shiny/T1Tarea1_OPT.git
   ```
<p align="right">(<a href="#readme-top">Volver al principio</a>)</p>

### Visualizar contenido

Abrir el archivo [index.html](https://github.com/little-shiny/T1Tarea1_OPT/blob/main/index.html) con el navegador 
web de su preferencia o utilizar VisualStudioCode con Live preview

## Características y mejoras implementadas

- Creación de formulario de contacto con HTML y CSS en index.html
- Vinculación mediante un botón y enlaces con otro archivo horarios-reuniones.html
- Archivo horarios-reuniones.html que contiene una tabla básica con un estilo sencillo en CSS del horario de las 
  diferentes tutorías del grado
